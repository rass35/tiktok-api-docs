Política de Privacidade

Última atualização: 25 de junho de 2026

Este aplicativo utiliza a API oficial do TikTok para autenticação e publicação de vídeos autorizados pelo usuário.

Coleta de Dados

O aplicativo pode acessar informações autorizadas pelo usuário através do processo de login do TikTok.

Uso das Informações

As informações são utilizadas exclusivamente para autenticação e envio de vídeos para a conta autorizada.

Compartilhamento de Dados

Nenhum dado pessoal é vendido, alugado ou compartilhado com terceiros.

Segurança

Os dados de autenticação são utilizados apenas para comunicação com os serviços oficiais do TikTok.

Contato

Em caso de dúvidas, entre em contato com o desenvolvedor do aplicativo.
